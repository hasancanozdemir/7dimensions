PTT Kargo Entegrasyonu

Odoo Teslimatlarınızı PTT Kargo ile Otomatikleştirin

Öne Çıkan Özellikler

Otomatik Barkod

Teslimat emrini onayladığınız anda PTT Kargo barkodu ve takip numarası otomatik olarak oluşturulur.

Kolay Kurulum

Sadece PTT API kullanıcı adı ve şifrenizi girerek saniyeler içinde kullanmaya başlayın.

Nasıl Çalışır?

1. Ayarlar > Stok > PTT Kargo menüsünden API şifrelerinizi girin.
2. Bir teslimat emri oluşturun ve onaylayın.
3. "PTT'ye Gönder" butonuna basın.
4. Barkod numaranız ekrana gelsin!
